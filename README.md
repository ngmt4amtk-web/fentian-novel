# 焚天 ─ 相国董卓伝

立ち絵付きノベルゲーム（FGO形式・恋愛ADV）。**あなた＝董卓**として、送り込まれた刺客・貂蝉を「落とす」。

- **媚び＝罠**：普通の恋愛ゲームの正解（優しさ・媚び）は貂蝉に軽蔑される。董卓らしい尊大さ・嘘のなさが正解。
- **4友好度**：貂蝉好感度／呂布嫉妬／李儒信頼／連環露見。選択肢で増減。
- **4エンド**：比翼（真・生存）／孤王（代償生存）／焚天（史実死）／落花（悲劇）。三国志演義（連環計）準拠。
